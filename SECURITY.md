## Reporting a Vulnerability

We take the security of our project seriously. If you discover a security vulnerability, please report it to us responsibly.

### How to Report

Please send an email to the maintainers at [security contact email]. Include as much detail as possible, such as:

- The type of issue (e.g., buffer overflow, SQL injection)
- Affected version(s)
- Steps to reproduce the issue
- Potential impact

### What to Expect

- **Acknowledgment**: You will receive an acknowledgment of your report within 48 hours.
- **Updates**: We will keep you informed about the progress toward a fix and any announcements.
- **Credit**: You will be credited in our security advisories unless you wish to remain anonymous.

### Scope

This policy covers all code within this repository. Please do not disclose the vulnerability publicly until we have had a chance to address it.
